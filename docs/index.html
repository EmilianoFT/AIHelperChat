AI Helper – Eclipse Update Site

This site hosts the AI Helper Eclipse plugin update repository. It is not intended for direct browsing.

To install or update the plugin, use Eclipse: 

Help → Install New Software…
Add → https://emilianoft.github.io/AIHelperChat

You can also update the plugin via: 

Help → Check for Updates

Quick Start

  1. Install AI Helper from this update site and restart Eclipse.
  2. Open the view: Window → Show View → Other… → AI Helper Chat.
  3. Configure your provider credentials in Preferences → AI Helper.
  4. Send a short prompt with Ctrl+Enter to validate the setup.

If the assistant does not answer, verify API key, selected model, and network access.
